# 📱 Projeto Celular

Simulação de um celular interativo feita com HTML e CSS, inspirada no modelo apresentado pelo professor Gustavo Guanabara.

---

## Linguagens utilizadas

- HTML5  
- CSS3  

---

## Conceitos aplicados

- Uso de `iframe`  
- Estrutura com `nav`  
- Centralização de elementos (`div`)  
- Navegação entre páginas locais  
- Animações em botões  

---

## 📌 Sobre o projeto

Este projeto simula a interface de um celular.  
No centro da tela, um `iframe` exibe diferentes páginas, que mudam conforme o usuário clica nos botões laterais.

O objetivo foi praticar conceitos fundamentais de HTML e CSS, focando em estruturação, navegação e estilização.

---
##  Acesse o projeto 
https://saulo-lima.github.io/projeto-celular/
---

## 🎮 Como funciona

- Os botões laterais funcionam como aplicativos  
- Ao clicar, o conteúdo do `iframe` é alterado  
- Tudo foi feito **sem JavaScript**, apenas com HTML e CSS
